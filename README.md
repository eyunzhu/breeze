### 函数：
- checkToggle()
	- 作用：单选按钮 全选 或 取消全选 [进入演示](http://breeze.eyunzhu.com/demo/js/checkToggle.html)
	
	``` 
	函数名：checkToggle
	参数：name （单选框 name属性）
	功能：全选 或 取消全选
	语法例：
	<button onclick="checkToggle('hobby')">全选</button>
	```
